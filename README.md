# Weekly Daily Tasks Timetable

This is a simple HTML-based weekly timetable for daily tasks and routines.

## How to View

Open `index.html` in any web browser to view the timetable.

## Description

The timetable displays a weekly schedule from Monday to Sunday, with time slots from 6:00 to 23:00. Each cell contains typical daily tasks or activities for that time period.

## Customization

To modify the timetable, edit the HTML table in `index.html`. You can change the tasks, times, or add more days/rows as needed. This can be adapted to your personal schedule.
